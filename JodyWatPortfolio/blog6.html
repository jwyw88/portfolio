Creating Graphs in Illustrator

12 April

  • A graph is a diagram of data that shows relationship among a set of numbers.
  • Data can be represented by a graphic element such as a:
    • Bar
    • Line
    • Point
  • Illustrator offers nine types of graphs:
    • Column
    • Stacked Column
    • Bar
    • Stacked Bar
    • Line
    • Area
    • Scatter
    • Pie
    • Radar
  • In illustrator you can convert one type of graph into another type and create custom designs
  • Each category labels in the first column of the Graph Data window
  • First row reserved for legend labels

For more information, please refer to Adobe for step-by-step procedure to create a graph
